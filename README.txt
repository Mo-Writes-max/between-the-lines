MO SHORT STORY WEBSITE

How to preview:
1. Open index.html in a web browser.

How to publish free with GitHub Pages:
1. Create a free GitHub account.
2. Create a new public repository.
3. Upload everything inside this folder.
4. Open Settings > Pages.
5. Choose "Deploy from a branch," select main and /root, then Save.

Important:
- The labels Light, Shadows, and Darkness are not shown anywhere on the public website.
- Replace the blank contact link later with a dedicated author email.
